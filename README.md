# Проект: Место

### Добро пожаловать на мой учебный проект :)

**О проекте**

Данный проект представляет страницу с возможностью изменения информации профля с лайв-валидацией, добавления и удаления фотографий и выставления лайков.

**Технологии**

* HTML5
* CSS3
* БЭМ
* Модули
* ООП
