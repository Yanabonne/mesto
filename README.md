# Проект: Место

### Добро пожаловать на мой учебный проект :)

**:v: О проекте**

Данный проект представляет страницу с возможностью изменения информации профиля с лайв-валидацией, добавления и удаления фотографий и выставления лайков.

**:v: Технологии**

- HTML5
- CSS3
- JS
- Webpack
- БЭМ
- Модули
- ООП
